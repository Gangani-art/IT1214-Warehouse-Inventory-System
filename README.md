# Warehouse Inventory System

## Project Description
This is a simple Java-based Warehouse Inventory System.
It allows users to:
- Add items
- View items
- Add stock
- Remove stock

## How to Run

1. Compile the program:
   javac *.java

2. Run the program:
   java Main
